Listas ordenadas

Ao usar a tag OL para fazer uma lista, a forma padrão vem assim:

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes
  5. Ir para a faculdade
  6. Almoçar
  7. Ir para o trabalho
  8. Voltar para casa
  9. Jantar
  10. Dormir

Mas tem dois parâmetros nessa tag que podem fazer modificações: o parâmetro TYPE e o parâmetro START.
No parâmetro Type temos cinco tipos: 1, A, a, I, i

Com o type="1" é o tipo padrão que ja vem na tag OL, a lista fica:

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes
  5. Ir para a faculdade
  6. Almoçar
  7. Ir para o trabalho
  8. Voltar para casa
  9. Jantar
  10. Dormir

Com o type="A" , a lista fica:

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes
  5. Ir para a faculdade
  6. Almoçar
  7. Ir para o trabalho
  8. Voltar para casa
  9. Jantar
  10. Dormir

No parâmetro START você escolhe qual "número" vai começar a contagem da lista. usa-se: start="numero que começará"

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes
  5. Ir para a faculdade
  6. Almoçar
  7. Ir para o trabalho
  8. Voltar para casa
  9. Jantar
  10. Dormir

Veja que a contagem da lista acima começa do "d" e não do "a" porque o start foi 4, ai ele começará da quarta letra.

Listas não ordenadas

são listas onde a ordem não é relevante. por exemplo, lista de compras. sua forma padrão é essa:

  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão

Mas na Unordered List(ul) também tem o parâmetro TYPE.
No parâmetro Type, na ul, temos três tipos: disc, circle, square

Com o type="disc" é o tipo padrão que ja vem na tag UL, a lista fica:

  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão

Com o type="circle" , a lista fica:

  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão

Com o type="square" , a lista fica:

  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão

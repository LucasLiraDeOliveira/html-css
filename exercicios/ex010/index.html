Isso é a tag A sem nenhum atributo extra (forma padrão).
Os atributos que podem ser agregados a tag A são:

hreflang
Permite identificar qual é o idioma principal do site para onde o link está desviando o fluxo da navegação. Isso permite avidar ao navegador e ao software de tradução como lidar caso o visitante opte por traduzir automaticamente os conteúdos.
target
Atributo que permite controlar onde o site de destino vai abrir:
_blank
Abre o link em uma nova janela em branco;
_self
Abre o link na janela, ou frame, atual (padrão);
rel
Recurso que indica qual é a natureza do destino. Ele tem vários valores:
next
Indica que o link é para a próxima parte do documento atual;
prev
Indica que o link é para a parte anterior do documento atual;
author
Indica que é um link para o site do autor do artigo atual;
external
Indica que é um link para outro site que não faz parte do site atual;
nofollow
Indica que é um link para um site não endossado como um link pago;
